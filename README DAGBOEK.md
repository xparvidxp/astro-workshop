Als een afbeelding decoratief is, kan de alt-text leeg. Hij moet wel altijd een alt-attribuut hebben.

- Wat maakt het voor hem/haar moeilijk.
-> Moeite met zien zorgt ervoor dat hij de website uitvergroot moet kijken. Dingen poppen up en maken de site voor hem moeilijker.
Websites te veel informatie, te vermoeiend, het moet snel bruikbaar zijn.

- Wat heeft hij/zij nodig?
Dat bijvoorbeeld de header verdwijnt en dat foto's groter worden bij groot inzoomen. Schaalbaarheid, contrast, gerelateerde informatie bij elkaar, visuele indicatoren zoals een focusrand.
